DESaccess Documentation
=====================================

DESaccess is a collection of apps and services from the `DES Labs <https://deslabs.ncsa.illinois.edu/>`_ team at the `National Center for Supercomputing Applications <https://www.ncsa.illinois.edu/>`_ that provides multiple tools you can use to access data from the `Dark Energy Survey <https://www.darkenergysurvey.org/>`_.
